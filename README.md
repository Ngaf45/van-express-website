# All Projects t/a Van Express Website

A professional, modern website for All Projects t/a Van Express (Pty) Ltd - a South African delivery and property services company.

## Features

✅ **Responsive Design** - Mobile-friendly layout that works on all devices  
✅ **Modern Styling** - Clean, professional design with green and orange branding  
✅ **Service Showcase** - Organized categorization of all services  
✅ **Contact Integration** - Quick quote request form with email integration  
✅ **Smooth Animations** - Fade-in effects and hover interactions  
✅ **Fast Loading** - Optimized with high-quality placeholder images  

## Color Scheme

- **Primary Color**: Dark Green (#1b5e20) - Represents landscaping and growth
- **Secondary Color**: Orange (#ff8c42) - Energy and professionalism
- **Background**: White with light grey sections
- **Buttons**: Green with orange hover effects

## Sections

1. **Navigation** - Sticky navbar with smooth scrolling
2. **Hero Section** - Eye-catching headline with CTAs
3. **About Us** - Company description and commitment
4. **Services** - Organized into 4 categories:
   - Transport & Logistics
   - Events
   - Landscaping & Garden Services
   - Property Maintenance
5. **Why Choose Us** - Six key advantages
6. **Contact** - Multiple contact methods and quote form
7. **Footer** - Company info and additional links

## Getting Started

1. Save all files in the same directory
2. Open `index.html` in your web browser
3. The website is fully functional and responsive

## Customization

### Update Contact Information
Edit the following in `index.html`:
- Phone number: `083 546 0678`
- Email: `stanley.thevanexpress@gmail.com`
- Business hours: `Monday – Saturday, 08:00 – 17:00`

### Change Images
Replace the Unsplash image URLs with your own:
- Search for `unsplash.com` URLs in the HTML and replace with your images
- Maintain the same image dimensions for consistency

### Modify Colors
Edit the CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #1b5e20;
    --secondary-color: #ff8c42;
    /* ... other colors */
}
```
